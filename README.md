# 🍎 AI Fruits Classifier

A Streamlit web app that classifies fruits as Fresh or Rotten using a Deep Learning model.

## 🚀 Features
- Upload image
- Predict fruit type
- Detect if it's fresh or rotten
- Confidence score display

## 🧠 Model
- MobileNetV2
- Trained on fruits dataset

## ▶️ Run locally
```bash
streamlit run app.py
